# Product Requirements Document (PRD)

Version: 1.0

Status: MVP

---

# Overview

EduPath is an AI-powered learning operating system that helps students retain knowledge over long periods through personalized tutoring, adaptive revision scheduling, persistent memory, and intelligent multi-agent orchestration.

The MVP is designed for the Kaggle × Google Vibe Coding Capstone and demonstrates how AI agents can work together to solve a meaningful educational problem.

---

# Problem Statement

Students preparing for competitive examinations often study consistently but struggle to retain previously learned concepts.

Existing learning platforms primarily focus on content delivery rather than long-term knowledge retention.

As a result:

* revision becomes inefficient
* students forget important concepts
* confidence decreases
* study planning becomes overwhelming

EduPath addresses these challenges through adaptive learning powered by persistent AI memory.

---

# Product Objectives

The MVP should demonstrate that an AI system can:

* remember previous learning sessions
* personalize future study sessions
* predict forgetting
* generate intelligent revision plans
* motivate consistent learning
* explain recommendations transparently

# Core Features (MVP)

## 1. AI Tutor

Provides concept explanations tailored to the student's current understanding.

---

## 2. Adaptive Recall Engine

Schedules revision based on:

* confidence
* forgetting probability
* importance
* historical performance

---

## 3. Learning Graph

Stores relationships between concepts.

Tracks:

* mastery
* dependencies
* confidence
* revision history

---

## 4. Persistent Memory

Maintains student history across sessions.

Stores:

* mistakes
* achievements
* preferences
* completed lessons
* weak topics

---

## 5. Quiz Generator

Creates adaptive quizzes using:

* concept mastery
* difficulty
* revision schedule

---

## 6. Study Planner

Generates personalized study schedules.

Considers:

* available time
* exam date
* weak topics
* revision workload

---

## 7. Progress Analytics

Visualizes:

* Memory Health
* Confidence
* Topic Mastery
* Weekly Progress
* Revision History

---

## 8. Gamification

Includes:

* XP
* Levels
* Badges
* Streaks
* Memory Recovery Rewards

---

## 9. Core Innovation

EduPath introduces the Cognitive Learning Engine (CLE).

The CLE combines:

• Long-Term Memory

• Knowledge Constellation

• Knowledge Recovery Engine

• Learning Analytics

to create a continuously evolving model of student understanding.

The following ideas are intentionally excluded from the first version:

* Parent dashboard
* Teacher dashboard
* OCR note scanning
* Voice tutor
* Multiplayer challenges

* Offline support

The system shall:

* authenticate users
* persist learning history
* maintain multiple study sessions

# Definition of MVP

The MVP is considered complete when a student can:

1. Sign in.

2. Learn a concept.

3. Receive personalized explanations.

4. Take an adaptive quiz.

5. Have the session remembered.

6. Return later.

7. Receive revision recommendations based on previous learning.

8. View progress analytics.

9. Earn rewards for consistent retention.
